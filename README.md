# Test Engine Ports

## Mods
 * Hypno's Lullaby
 * Smoke Em Out Struggle
 * Sonic.exe
 
## How to install
Heres a tutorial on how to install these mods

1. Download the source code
2. Copy the folders
3. Paste the folders into Test Engines Mods folder
4. Open Test Engine
5. Goto Main Menu and press M
6. Select your mod
7. Done!
